Página principal creada para practicar HTML y CSS.

Sobre este proyecto

Este pequeño sitio web presenta la idea general de mi proyecto personal. Su objetivo es mostrar cómo puedo estructurar contenido usando HTML y luego personalizar su apariencia con hojas de estilo en CSS.

Puntos destacados

  • Organización clara del contenido en secciones.
  • Uso de colores y tipografías para mejorar la lectura.
  • Diseño adaptable para pantallas grandes y pequeñas.
